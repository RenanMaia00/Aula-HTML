HTML

HTML é a linguagem responsável pela estrutura das páginas web.

Com ele definimos títulos, parágrafos, imagens, links e organização do conteúdo.

Saiba mais em: MDN Web Docs

CSS

CSS é utilizado para estilizar páginas web.

Permite alterar cores, espaçamentos, tamanhos e layouts.

JavaScript

JavaScript permite adicionar interatividade às páginas web.
